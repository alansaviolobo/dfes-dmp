PDF Layout Template

This template shows the layout structure for the PDF export footer. The layout uses a three-column flex design:

  • Left: QR code (12mm × 12mm) and date
  • Center: Title, description, data sources, and URL (wraps as needed)
  • Right: Scale bar and north arrow

Key Measurements:

  • Footer box padding: 4mm
  • Gap between columns: 4mm
  • Gap between text elements: 1.2mm
  • Title line height: 3.5mm
  • Regular text line height: 2.2mm
  • QR code size: 12mm × 12mm
  • Scale bar area: ~30mm width
  • North arrow size: 6mm

Text Wrapping: All text elements in the center column can wrap to prevent overflow. The box width is calculated dynamically based on content, with a maximum width constraint.
